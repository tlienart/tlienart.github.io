Matrix inversion lemmas

The Woodbury formula is maybe one of the most ubiquitous trick in basic linear algebra: it starts with the explicit formula for the inverse of a block 2x2 matrix and results in identities that can be used in kernel theory, the Kalman filter, to combine multivariate normals etc.

In these notes I present a simple development leading to the Woodbury formula, and the special case of the Sherman-Morrison formula with some code to show those at work.

Partitioned matrix inversion

Basic lemmas

Equating the expressions in (4) and (6) for YYY gives D−1CW=ZCA−1D^{-1} CW = ZCA^{-1}D−1CW=ZCA−1 which, combined with (5) gives the first lemma.

(Lemma I) let AAA and DDD be square, invertible matrices of size nA×nAn_A\times n_AnA​×nA​ and nD×nDn_D\times n_DnD​×nD​ and BBB and CCC be matrices of size nA×nDn_A\times n_DnA​×nD​ and nD×nAn_D\times n_AnD​×nA​, the following identity holds:

D−1C(A−BD−1C)−1 ⁣ ⁣= ⁣ ⁣(D−CA−1B)−1CA−1. D^{-1} C (A-BD^{-1} C)^{-1} \quad\!\! =\quad\!\! (D - CA^{-1} B)^{-1} CA^{-1}. D−1C(A−BD−1C)−1=(D−CA−1B)−1CA−1.

Equating the expressions in (4) and (6) for XXX gives WBD−1=A−1BZWBD^{-1} = A^{-1} B Z WBD−1=A−1BZ which, combined with (5) gives the second lemma.

(Lemma II) under the same assumptions as for Lemma I, the following identity holds:

(A−BD−1C)−1BD−1 ⁣ ⁣= ⁣ ⁣A−1B(D−CA−1B)−1. (A - BD^{-1} C)^{-1} BD^{-1} \quad\!\! =\quad\!\! A^{-1} B (D - CA^{-1} B)^{-1}. (A−BD−1C)−1BD−1=A−1B(D−CA−1B)−1.

Woodbury formula

One little bit of dark magic is required to get the Woodbury formula: observe that if we take the term (A−BD−1C)(A-BD^{-1} C)(A−BD−1C) and right-multiply it by −A−1-A^{-1}−A−1 we get

(A−BD−1C)(−A−1) ⁣ ⁣= ⁣ ⁣BD−1CA−1−I (A-BD^{-1} C)(-A^{-1}) \quad\!\! =\quad\!\! \textcolor{green}{BD^{-1}} \textcolor{blue}{CA^{-1}} - \mathbf I (A−BD−1C)(−A−1)=BD−1CA−1−I

and therefore BD−1CA−1=(I+(A−BD−1C)(−A−1))BD^{-1} CA^{-1} = (\mathbf I + (A-BD^{-1} C)(-A^{-1}))BD−1CA−1=(I+(A−BD−1C)(−A−1)). Now if we post-multiply (8) by CA−1CA^{-1}CA−1 and re-arrange the expression, we get the third lemma.

(Lemma III) under the same assumptions as for Lemma I, the following identity holds:

(A−BD−1C)−1 ⁣ ⁣= ⁣ ⁣A−1+A−1B(D−CA−1B)−1CA−1. (A-BD^{-1} C)^{-1} \quad\!\! =\quad\!\! A^{-1} + A^{-1} B(D-CA^{-1} B)^{-1} CA^{-1}. (A−BD−1C)−1=A−1+A−1B(D−CA−1B)−1CA−1.

of course the same gymnastics can be applied with the term (D−CA−1B)−1(D-CA^{-1} B)^{-1}(D−CA−1B)−1 to obtain a similar identity.

To obtain the classical Woodbury formula though, we just need to reassign letters with E←AE\leftarrow AE←A, F←−BF\leftarrow -BF←−B, G←D−1G\leftarrow D^{-1}G←D−1 and H←CH\leftarrow CH←C. (So Lemma III is already the Woodbury formula, the re-assignment only leads to a somewhat more visually pleasing form)

(Woodbury formula) let EEE, GGG be square invertible matrices of dimensions nE×nEn_E \times n_EnE​×nE​ and nG×nGn_G\times n_GnG​×nG​ respectively, let FFF and HHH be matrices of size nE×nGn_E\times n_GnE​×nG​ and nG×nEn_G\times n_EnG​×nE​ respectively, then the following identity holds:

(E+FGH)−1 ⁣ ⁣= ⁣ ⁣E−1−E−1F(G−1+HE−1F)−1HE−1 (E+FGH)^{-1} \quad\!\! =\quad\!\! E^{-1} - E^{-1} F(G^{-1} + HE^{-1} F)^{-1} H E^{-1} (E+FGH)−1=E−1−E−1F(G−1+HE−1F)−1HE−1

Sherman-Morrison formula

Consider again (11) and let G=1G=1G=1, F=uF=uF=u and H=vH=vH=v with u,v∈RnEu, v\in\mathbb R^{n_E}u,v∈RnE​ then the formula gives

(E+uvT)−1 ⁣ ⁣= ⁣ ⁣E−1−E−1uvTE−11+vTE−1u, (E+uv^T)^{-1} \quad\!\! =\quad\!\! E^{-1} -{E^{-1} u v^T E^{-1}\over 1 + v^T E^{-1} u}, (E+uvT)−1=E−1−1+vTE−1uE−1uvTE−1​,

a useful expression for the inverse of a matrix combined with a rank-1 perturbation. This is used for instance in the development of the famous BFGS flavour of the Quasi-Newton iterations (see e.g. the wikipedia article).

Revisiting the assumptions

thanks to Christopher Yeh for the interesting discussion on this topic. Chris also wrote a post on this topic.

In the development above, we made a few assumptions to simplify the development, sometimes stronger than needed. We can now make those more precise without risking confusion.

Let us start with the same description of MMM as in (1); we had introduced the Schur-Complement of AAA as As:=(D−CA−1B)A^s := (D - CA^{-1} B)As:=(D−CA−1B), and that of DDD as Ds:=(A−BD−1C)D^s := (A - BD^{-1} C)Ds:=(A−BD−1C).

With those definitions, we have the following set of properties:

(Theorem) Let MMM be as in (1):

  • (1A) if both MMM and AAA are invertible then AsA^sAs is invertible,

  • (1B) if both MMM and DDD are invertible then DsD^sDs is invertible,

  • (2A) if AAA and AsA^sAs are invertible then MMM is invertible,

  • (2B) if DDD and DsD^sDs are invertible then MMM is invertible,

  • (3) if both AAA and DDD are invertible as well as one of {M,As,Ds}\{M, A^s, D^s\}{M,As,Ds}, then they all are.

(1A - proof) take z=(x1,x2)z = (x_1, x_2)z=(x1​,x2​) a vector of dimension compatible with MMM and such that Asx2=0A^s x_2 = 0Asx2​=0 so that Dx2=CA−1Bx2Dx_2 = CA^{-1} B x_2Dx2​=CA−1Bx2​. Then, considering MzMzMz gives:

Mz=(Ax1+Bx2,Cx1+CA−1Bx2) Mz = (Ax_1 + Bx_2, Cx_1 + CA^{-1} B x_2) Mz=(Ax1​+Bx2​,Cx1​+CA−1Bx2​)

since AAA is invertible, we can form x1=−A−1Bx2x_1 = -A^{-1} B x_2x1​=−A−1Bx2​ so that Mz=0Mz=0Mz=0. Since MMM is invertible, z=0z=0z=0 and thus necessarily x2=0x_2=0x2​=0 so that AsA^sAs is invertible. Proof of 1B is identical.

(2A - proof) let z=(x1,x2)z=(x_1, x_2)z=(x1​,x2​) such that Mz=0Mz=0Mz=0 then

Ax1+Bx2=0Cx1+Dx2=0\begin{array}{rcl} Ax_1 + Bx_2 &=& 0\\ Cx_1 + Dx_2 &=& 0\end{array}Ax1​+Bx2​Cx1​+Dx2​​==​00​

since AAA is invertible, we can write x1=−A−1Bx2x_1 = -A^{-1} Bx_2x1​=−A−1Bx2​ and the second equation becomes

(D−CA−1B)x2 ⁣ ⁣= ⁣ ⁣0 (D - CA^{-1} B)x_2 \quad\!\! =\quad\!\! 0 (D−CA−1B)x2​=0

or Asx2=0A^s x_2 = 0Asx2​=0. But since AsA^sAs is invertible, x2=0x_2=0x2​=0 and therefore x1=0x_1=0x1​=0 so that zzz is necessarily 000 and MMM is invertible. Proof of 2B is identical.

(3 - proof) this is trivially implied by combining the previous properties.

Back to the development

In the development we were working under (3) with AAA, DDD and MMM invertible (and therefore AsA^sAs and DsD^sDs as well). We had then made the assumption that (I−A−1BC)(\mathbf I-A^{-1} BC)(I−A−1BC) and (I−D−1CA−1)(\mathbf I - D^{-1} CA^{-1})(I−D−1CA−1) were invertible, but these are actually implied by the fact that respectively DsD^sDs and AsA^sAs are invertible. Indeed, taking the first one, if we take zzz such that

(I−A−1BD−1C)z ⁣ ⁣= ⁣ ⁣0 (\mathbf I-A^{-1} B D^{-1} C)z \quad\!\! =\quad\!\! 0 (I−A−1BD−1C)z=0

and left-multiply by AAA, we have

(A−BD−1C)z ⁣ ⁣= ⁣ ⁣Dsz ⁣ ⁣= ⁣ ⁣0 (A - BD^{-1} C)z \quad\!\! =\quad\!\! D^sz \quad\!\! = \quad\!\! 0 (A−BD−1C)z=Dsz=0

but since DsD^sDs is invertible (by 3), z=0z=0z=0 so that (I−A−1BD−1C)(\mathbf I-A^{-1} BD^{-1} C)(I−A−1BD−1C) is invertible (the second case is identical).

A bit of code

If you want to see these equations at work, here's a simple Julia script:

using Test
using LinearAlgebra: dot

# Woodbury formula
n_E, n_G = 13, 15;
E = randn(n_E, n_E);
F = randn(n_E, n_G);
G = randn(n_G, n_G);
H = randn(n_G, n_E);
iE, iG = inv(E), inv(G);
@test inv(E+F*G*H) ≈ iE - iE*F*inv(iG + H*iE*F)*H*iE

# Sherman-Morrison formula
n_E = 23;
E = randn(n_E, n_E);
u = randn(n_E);
v = randn(n_E);
iE = inv(E)
iEu = iE*u
@test inv(E + u*v') ≈ iE - (iEu*(v'*iE))/(1+dot(v, iEu))

(Recall that invertible matrices are dense among square matrices so that using randomly generated matrices for EEE and GGG is unlikely to cause problems).

Tim Holy has written a simple package for this called WoodburyMatrices.jl.

using WoodburyMatrices
W = Woodbury(E, F, G, H);
b = randn(n_E);
# using the package
s1 = W\b;
# hand-coding using the formula
iEb = iE*b;
s2  = iEb - iE*(F*((iG+H*iE*F)\(H*iEb)))
@test s1 ≈ s2
