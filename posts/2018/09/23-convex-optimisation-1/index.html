Subgradient, subdifferential and FOC

We say that y∈Rny\in\mathbb R^ny∈Rn is a subgradient of the convex function fff at x∈Cx\in Cx∈C if it verifies the following inequality:

f(z)≥f(x)+⟨z−x,y⟩,∀z∈C.\begin{array}{rcl} f(z) &\ge & f(x) + \langle z-x, y \rangle, \qquad \forall z\in C. \end{array}f(z)​≥​f(x)+⟨z−x,y⟩,∀z∈C.​

The inequality (2) simply indicates that the graph of the function fff at xxx is supported by the hyperplane defined by the right-hand side. A subgradient is thus the "slope" of one such supporting hyperplane.

The function is differentiable at xxx if and only if there is a unique subgradient at xxx (the classical gradient ∇f(x)\nabla f(x)∇f(x)) and, correspondingly, only one supporting hyperplane:

f(z)≥f(x)+⟨z−x,∇f(x)⟩,∀z∈C.\begin{array}{rcl} f(z) &\ge& f(x) + \left\langle z-x, \nabla f(x)\right\rangle, \qquad \forall z \in C.\end{array}f(z)​≥​f(x)+⟨z−x,∇f(x)⟩,∀z∈C.​

However, if the function is not differentiable at xxx (e.g., if there is a kink at xxx) then there may be infinitely many supporting hyperplanes and infinitely many subgradients.

The set of subgradients of a convex function fff at a point x∈dom fx\in \mathrm{dom}\, fx∈domf is called the subdifferential of fff and denoted ∂f(x)\partial f(x)∂f(x). For a proper convex function fff, it can be shown that the subdifferential of fff is a non-empty bounded set at any point x∈(dom f)∘x\in (\mathrm{dom}\,f)^\circx∈(domf)∘ (Rockafellar (1970), theorem 23.4).

Note that since we have assumed that C⊆dom fC\subseteq \mathrm{dom}\,fC⊆domf, then C∘⊆(dom f)∘C^\circ\subseteq (\mathrm{dom}\,f)^\circC∘⊆(domf)∘ and therefore ∂f\partial f∂f is non-empty and bounded on C∘C^\circC∘.

An example is the absolute value function f(x)=∣x∣f(x)=|x|f(x)=∣x∣ which is not differentiable at 000. It is however supported at that point by all lines of the form ℓα(x)=αx\ell_\alpha(x)=\alpha xℓα​(x)=αx with α∈[−1,1]\alpha\in [-1,1]α∈[−1,1] (see the dashed lines on the figure below).
