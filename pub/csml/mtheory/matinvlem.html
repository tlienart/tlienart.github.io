Matrix inversion lemmas

The Woodbury formula is maybe one of the most ubiquitous trick in basic linear algebra: it starts with the explicit formula for the inverse of a block 2x2 matrix and results in identities that can be used in kernel theory, the Kalman filter, to combine multivariate normals etc.

In these notes I present a simple development leading to the Woodbury formula, and the special case of the Sherman-Morrison formula with some code to show those at work.

Woodbury formula

One little bit of dark magic is required to get the Woodbury formula: observe that if we take the term (A−BD−1C)(A-BD^{-1} C)(A−BD−1C) and right-multiply it by −A−1-A^{-1}−A−1 we get

(A−BD−1C)(−A−1) ⁣ ⁣= ⁣ ⁣BD−1CA−1−I (A-BD^{-1} C)(-A^{-1}) \quad\!\! =\quad\!\! \textcolor{green}{BD^{-1}} \textcolor{blue}{CA^{-1}} - \mathbf I (A−BD−1C)(−A−1)=BD−1CA−1−I

and therefore BD−1CA−1=(I+(A−BD−1C)(−A−1)BD^{-1} CA^{-1} = (\mathbf I + (A-BD^{-1} C)(-A^{-1})BD−1CA−1=(I+(A−BD−1C)(−A−1) . Now if we post-multiply (8) by CA−1CA^{-1}CA−1 and re-arrange the expression, we get the third lemma.

(Lemma III) under the same assumptions as for Lemma I, the following identity holds: (A−BD−1C)−1 ⁣ ⁣= ⁣ ⁣A−1+A−1B(D−CA−1B)−1CA−1. (A-BD^{-1} C)^{-1} \quad\!\! =\quad\!\! A^{-1} + A^{-1} B(D-CA^{-1} B)^{-1} CA^{-1}. (A−BD−1C)−1=A−1+A−1B(D−CA−1B)−1CA−1.

of course the same gymnastics can be applied with the term (D−CA−1B)−1(D-CA^{-1} B)^{-1}(D−CA−1B)−1 to obtain a similar identity. To obtain the classical Woodbury formula though, we just need to reassign letters with E←AE\leftarrow AE←A , F←−BF\leftarrow -BF←−B , G←D−1G\leftarrow D^{-1}G←D−1 and H←CH\leftarrow CH←C . (So Lemma III is already the Woodbury formula, the re-assignment only leads to a somewhat more visually pleasing form)

Sherman-Morrison formula

Consider again (11) and let G=1G=1G=1 , F=uF=uF=u and H=vH=vH=v with u,v∈RnEu, v\in\mathbb R^{n_E}u,v∈RnE​ then the formula gives

(E+uvT)−1 ⁣ ⁣= ⁣ ⁣E−1−E−1uvTE−11+vTE−1u, (E+uv^T)^{-1} \quad\!\! =\quad\!\! E^{-1} -{E^{-1} u v^T E^{-1}\over 1 + v^T E^{-1} u}, (E+uvT)−1=E−1−1+vTE−1uE−1uvTE−1​,

a useful expression for the inverse of a matrix combined with a rank-1 perturbation. This is used for instance in the development of the famous BFGS flavour of the Quasi-Newton iterations (see e.g. the wikipedia article).

A bit of code

If you want to see these equations at work, here's a simple Julia script:

using Test
# Woodbury formula
n_E, n_G = 13, 15;
E = randn(n_E, n_E);
F = randn(n_E, n_G);
G = randn(n_G, n_G);
H = randn(n_G, n_E);
iE, iG = inv(E), inv(G);
@test inv(E+F*G*H) ≈ iE - iE*F*inv(iG + H*iE*F)*H*iE

# Sherman-Morrison formula
n_E = 23;
E = randn(n_E, n_E);
u = randn(n_E);
v = randn(n_E);
iE = inv(E)
iEu = iE*u
@test inv(E + u*v') ≈ iE - (iEu*(v'*iE))/(1+dot(v, iEu))

(Recall that invertible matrices are dense among square matrices so that using randomly generated matrices for EEE and GGG is unlikely to cause problems).

Interestingly Tim Holy has written a simple package for this called WoodburyMatrices.jl.

using WoodburyMatrices
W = Woodbury(E, F, G, H);
b = randn(n_E);
# using the package
s1 = W\b;
# hand-coding using the formula
iEb = iE*b;
s2  = iEb - iE*(F*((iG+H*iE*F)\(H*iEb)))
@test s1 ≈ s2
