Convex analysis (part 3)

Strict and strong convexity

Remember that a function ψ:C→R\psi:C\to \mathbb Rψ:C→R is convex if the following inequality holds for any two x,z∈Cx, z \in Cx,z∈C and 0<λ<10<\lambda<10<λ<1: ψ((1−λ)x+λz)≤(1−λ)ψ(x)+λψ(z).\begin{array}{c} \psi((1-\lambda)x+\lambda z) &\le & (1-\lambda)\psi(x) + \lambda \psi(z). \end{array}ψ((1−λ)x+λz)​≤​(1−λ)ψ(x)+λψ(z).​ When that inequality holds strictly for any two x≠zx\neq zx​=z in CCC, the function ψ\psiψ is called strictly convex.

Let x∈C∘x\in C^\circx∈C∘ and y∈∂ψ(x)y\in \partial \psi(x)y∈∂ψ(x), ψ\psiψ is strictly convex on C∘C^\circC∘ if and only if the subgradient inequality holds strictly for all z≠xz\neq xz​=x: ψ(z)>ψ(x)+⟨z−x,y⟩,∀z∈C\{x}.\begin{array}{c} \psi(z) &>& \psi(x) + \left\langle z-x, y\right\rangle, \quad\forall z\in C\backslash\{x\}.\end{array}ψ(z)​>​ψ(x)+⟨z−x,y⟩,∀z∈C\{x}.​

This simply means that the graph of the function is strictly above its supporting hyperplanes.

We can prove this quickly by noting that (1) can be rearranged into ψ(z)>ψ(x)+ψ((1−λ)x+λz)−ψ(x)λ,\begin{array}{c} \psi(z) &>&\psi(x) + {\psi((1-\lambda)x+\lambda z) - \psi(x)\over \lambda}, \end{array}ψ(z)​>​ψ(x)+λψ((1−λ)x+λz)−ψ(x)​,​ now if we let y∈∂ψ(x)y\in\partial \psi(x)y∈∂ψ(x) and using the subgradient inequality, we can also write ψ((1−λ)x+λz)≥ψ(x)+λ⟨z−x,y⟩.\begin{array}{c} \psi((1-\lambda)x+\lambda z) &\ge& \psi(x) + \lambda\left\langle z-x, y\right\rangle.\end{array}ψ((1−λ)x+λz)​≥​ψ(x)+λ⟨z−x,y⟩.​ Combining (3) and (4) gives (2).

Example: ψ(x)=xlog⁡(x)\psi(x)=x\log(x)ψ(x)=xlog(x) is strictly convex on R+\mathbb R^+R+ whereas ψ(x)=∣x∣\psi(x)=|x|ψ(x)=∣x∣ is not.

The function φ:C→R\varphi:C\to \mathbb Rφ:C→R is said to be μ\muμ-strongly convex at x∈Cx\in Cx∈C with parameter μ>0\mu>0μ>0 if the subgradient inequality holds with:

φ(z)≥φ(x)+⟨z−x,y⟩+μ2∥z−x∥22,\begin{array}{c} \varphi(z) &\ge& \varphi(x) + \left\langle z-x, y\right\rangle + {\mu\over 2}\|z-x\|_2^2,\end{array}φ(z)​≥​φ(x)+⟨z−x,y⟩+2μ​∥z−x∥22​,​

for any z∈Cz\in Cz∈C and y∈∂φ(x)y\in\partial \varphi(x)y∈∂φ(x).

Intuitively, a strongly convex function is a strictly convex function whose graph goes away from the supporting hyperplane sufficiently fast (at least quadratically fast).

Strict convexity and differentiability

Let ψ:C→R\psi:C\to \mathbb Rψ:C→R a convex function; ψ\psiψ is strictly convex on C∘C^\circC∘ if and only if ψ⋆\psi^\starψ⋆ is differentiable on ∂f(C∘)={y∈Rn ∣ ∃x∈C∘,y∈∂f(x)}\partial f(C^\circ) = \{y\in\mathbb R^n\,|\,\exists x\in C^\circ, y\in\partial f(x)\}∂f(C∘)={y∈Rn∣∃x∈C∘,y∈∂f(x)}.

To prove this result, we can use the link between ∂ψ\partial\psi∂ψ and ∂ψ⋆\partial\psi^{\star}∂ψ⋆ (cf. convex analysis part 2): x∈∂f(y)∩C⟺(y∈∂f(x),x∈C).\begin{array}{c} x\in \partial f(y)\cap C &\Longleftrightarrow& (y\in\partial f(x), x\in C). \end{array}x∈∂f(y)∩C​⟺​(y∈∂f(x),x∈C).​ ⇒\Rightarrow⇒: assume f⋆f^\starf⋆ is not differentiable on C∘C^\circC∘. Then, there is a yyy such that ∂f⋆(y)\partial f^\star(y)∂f⋆(y) contains more than one points, say x1≠x2x_1\neq x_2x1​​=x2​ in in C∘C^\circC∘. But then, using the equivalence (6) we have that y∈∂f(x1)∩∂f(x2)y\in\partial f(x_1) \cap \partial f(x_2)y∈∂f(x1​)∩∂f(x2​). Since fff is strictly convex and x1≠x2x_1\neq x_2x1​​=x2​ the two following strict inequalities must hold: f(x2) ⁣ ⁣> ⁣ ⁣f(x1)+⟨x2−x1,y⟩,f(x1) ⁣ ⁣> ⁣ ⁣f(x2)+⟨x1−x2,y⟩.\begin{aligned} f(x_2)\quad\!\! >\quad\!\!f(x_1)+\left\langle x_2-x_1, y\right\rangle,\\ f(x_1)\quad\!\! >\quad\!\!f(x_2)+\left\langle x_1-x_2, y\right\rangle.\end{aligned}f(x2​)>f(x1​)+⟨x2​−x1​,y⟩,f(x1​)>f(x2​)+⟨x1​−x2​,y⟩.​ But summing both leads to a contradiction.

⇐\Leftarrow⇐: assume f⋆f^\starf⋆ is differentiable and take a yyy with x=∇f⋆(y)x=\nabla f^\star(y)x=∇f⋆(y) in C∘C^\circC∘. Let's now assume that fff is not strictly convex on C∘C^\circC∘. Then, there exists z∈C∘z\in C^\circz∈C∘ such that f(z)=f(x)+⟨z−x,y⟩f(z)=f(x)+\left\langle z-x, y\right\ranglef(z)=f(x)+⟨z−x,y⟩. But this can also be written f(x)=f(z)+⟨x−z,y⟩f(x)=f(z)+\left\langle x-z,y\right\ranglef(x)=f(z)+⟨x−z,y⟩ so that y∈∂f(z)y\in\partial f(z)y∈∂f(z) by definition of the subgradient. This means that both xxx and zzz are in ∂f⋆(y)\partial f^\star(y)∂f⋆(y) which contradicts the differentiability of f⋆f^\starf⋆.

Bregman divergence

Consider the case where ψ\psiψ is strictly convex and differentiable on C∘C^\circC∘ so that ∂ψ(x)={∇ψ(x)}\partial \psi(x)=\{\nabla \psi(x)\}∂ψ(x)={∇ψ(x)} for x∈C∘x\in C^\circx∈C∘. In that case, we have ψ(z)≥f(x)+⟨z−x,∇ψ(x)⟩,∀z∈C,\begin{array}{c} \psi(z) &\ge& f(x) + \left\langle z-x, \nabla \psi(x)\right\rangle, \quad \forall z\in C,\end{array}ψ(z)​≥​f(x)+⟨z−x,∇ψ(x)⟩,∀z∈C,​ with equality if and only if z=xz=xz=x. We can use this to define a notion of similarity between two points xxx and zzz in CCC.

Let ψ\psiψ denote a strictly convex and differentiable function on C∘C^\circC∘. The Bregman-divergence on C∘×C∘C^\circ\times C^\circC∘×C∘ associated with ψ\psiψ is defined as Bψ(z,x)=ψ(z)−ψ(x)−⟨z−x,∇ψ(x)⟩.\begin{array}{c} B_\psi(z, x) &=& \psi(z)-\psi(x)-\left\langle z-x,\nabla \psi(x)\right\rangle.\end{array}Bψ​(z,x)​=​ψ(z)−ψ(x)−⟨z−x,∇ψ(x)⟩.​

Given (8), we have that Bψ(z,x)>0B_\psi(z, x)>0Bψ​(z,x)>0 for all z≠xz\neq xz​=x and Bψ(z,x)=0B_\psi(z, x)=0Bψ​(z,x)=0 iff x=zx=zx=z which shows that BψB_\psiBψ​ is a valid divergence (positive definite functional).

For example, if we take ψ(x)=12⟨x,x⟩\psi(x) = \frac12\left\langle x, x\right\rangleψ(x)=21​⟨x,x⟩ then Bψ(z,x) ⁣ ⁣= ⁣ ⁣12⟨z,z⟩−12⟨x,x⟩−⟨z−x,x⟩= ⁣ ⁣12⟨z−x,z−x⟩\begin{aligned} B_\psi(z, x) \quad\!\!&=\quad\!\! \frac12\left\langle z, z\right\rangle - \frac12\left\langle x, x\right\rangle - \left\langle z-x, x\right\rangle\\ &=\quad\!\! \frac12\left\langle z-x, z-x\right\rangle\end{aligned}Bψ​(z,x)​=21​⟨z,z⟩−21​⟨x,x⟩−⟨z−x,x⟩=21​⟨z−x,z−x⟩​ which is just the squared Euclidean (ℓ2\ell^{2}ℓ2) distance between zzz and xxx. The factor 1/21/21/2 might seem irrelevant but it makes other developments a bit nicer so that, usually in convex-optimisation, the squared Euclidean distance is scaled by a factor two.

Note that if the function φ\varphiφ is μ\muμ-strongly convex and differentiable, then the Bregman divergence associated to it is lower bounded by the squared ℓ2\ell^{2}ℓ2-distance:

Bφ(z,x)≥μ2∥z−x∥22,∀x,z∈C.\begin{array}{c} B_{\varphi}(z,x) &\ge& {\mu\over 2} {\|z-x\|^{2}_{2}}, \quad\forall x,z \in C. \end{array}Bφ​(z,x)​≥​2μ​∥z−x∥22​,∀x,z∈C.​

Bregman divergence and convex conjugacy

Let ψ\psiψ be a differentiable and strictly convex function on C∘C^\circC∘ then ψ⋆\psi^\starψ⋆ is also differentiable and strictly convex on ∇ψ(C∘)\nabla \psi(C^\circ)∇ψ(C∘) and we can consider the Bregman divergence that it induces.

Bψ⋆(u,v)=ψ⋆(u)−ψ⋆(v)−⟨u−v,∇ψ⋆(v)⟩.\begin{array}{c} B_{\psi^\star}(u, v) &=& \psi^\star(u) - \psi^\star(v) - \left\langle u-v, \nabla\psi^\star(v)\right\rangle.\end{array}Bψ⋆​(u,v)​=​ψ⋆(u)−ψ⋆(v)−⟨u−v,∇ψ⋆(v)⟩.​

Let us consider a pair x,z∈C∘x, z\in C^\circx,z∈C∘ with u=∇ψ(x)u=\nabla\psi(x)u=∇ψ(x) and v=∇ψ(z)v=\nabla\psi(z)v=∇ψ(z). Then, since (x,u)(x, u)(x,u) and (z,v)(z, v)(z,v) are dual pairs, we have that ψ⋆(u)+ψ(x)=⟨x,u⟩\psi^\star(u)+\psi(x)=\left\langle x, u\right\rangleψ⋆(u)+ψ(x)=⟨x,u⟩ (and the same in (z,v)(z, v)(z,v)) as well as ∇ψ⋆(v)=z\nabla\psi^\star(v)=z∇ψ⋆(v)=z so that Bψ⋆(u,v)B_{\psi^\star}(u, v)Bψ⋆​(u,v) simplifies to Bψ⋆(u,v)=ψ(z)−ψ(x)−⟨z−x,∇ψ(x)⟩,\begin{array}{c} B_{\psi^\star}(u, v) &=& \psi(z) - \psi(x) - \left\langle z-x, \nabla \psi(x)\right\rangle,\end{array}Bψ⋆​(u,v)​=​ψ(z)−ψ(x)−⟨z−x,∇ψ(x)⟩,​ which proves the following result.

Let ψ\psiψ be a differentiable and strictly convex function on C∘C^\circC∘ then the following equality holds: Bψ⋆(∇ψ(x),∇ψ(z))=Bψ(z,x),\begin{array}{c} B_{\psi^\star}(\nabla\psi(x), \nabla\psi(z)) &=& B_\psi(z, x),\end{array}Bψ⋆​(∇ψ(x),∇ψ(z))​=​Bψ​(z,x),​ for any x,z∈C∘x,z \in C^\circx,z∈C∘.

Observe how the arguments get "swapped".

Lipschitz continuity and strong convexity

This is a useful result to prove convergence rates of some iterative minimisation methods but can easily be skipped in a first reading.

We will show that if φ\varphiφ is a differentiable, strongly convex function on C∘C^\circC∘ then ∇φ⋆\nabla \varphi^\star∇φ⋆ is Lipschitz continuous on ∇φ(C∘)\nabla\varphi(C^\circ)∇φ(C∘). Recall that a function ϕ\phiϕ is said to be β\betaβ-Lipschitz-continuous on E⊆dom ϕE\subseteq \mathrm{dom}\,\phiE⊆domϕ if the following inequality holds ∀u,v∈E\forall u,v\in E∀u,v∈E:

∥ϕ(u)−ϕ(v)∥2≤β∥u−v∥2.\begin{array}{c} \|\phi(u)-\phi(v)\|_2 &\le& \beta \|u-v\|_2.\end{array}∥ϕ(u)−ϕ(v)∥2​​≤​β∥u−v∥2​.​

Using (11), we can write μ∥x−z∥22≤Bφ(x,z)+Bφ(z,x).\begin{array}{c} \mu \|x-z\|^{2}_{2} &\le& B_{\varphi}(x,z)+B_{\varphi}(z,x) .\end{array}μ∥x−z∥22​​≤​Bφ​(x,z)+Bφ​(z,x).​ for any x,z∈C∘x, z\in C^\circx,z∈C∘. Plugging the definition of the Bregman divergence in the right hand side and letting u=∇φ(x)u=\nabla \varphi(x)u=∇φ(x) and v=∇φ(z)v=\nabla \varphi(z)v=∇φ(z) we get μ∥x−z∥22 ⁣ ⁣≤ ⁣ ⁣⟨z−x,u−v⟩ ⁣ ⁣≤ ⁣ ⁣∥x−z∥2∥u−v∥2,\begin{aligned} \mu\| x-z\|^{2}_{2} &\quad\!\! \le\quad\!\! \langle z-x, u-v \rangle \\ &\quad\!\! \le\quad\!\! \|x-z\|_2\|u-v\|_2,\end{aligned}μ∥x−z∥22​​≤⟨z−x,u−v⟩≤∥x−z∥2​∥u−v∥2​,​ using Cauchy-Schwartz's inequality on the second line. Rearranging terms yields ∥x−z∥2≤1μ∥u−v∥2.\begin{array}{c} \|x-z\|_{2} &\le& {1\over \mu}\| u-v\|_2.\end{array}∥x−z∥2​​≤​μ1​∥u−v∥2​.​

Since x∈C∘x\in C^\circx∈C∘ and u=∇φ(x)u=\nabla \varphi(x)u=∇φ(x), we can write x=∇φ⋆(u)x = \nabla \varphi^\star(u)x=∇φ⋆(u) and similarly, z=∇φ⋆(v)z=\nabla\varphi^\star(v)z=∇φ⋆(v) (see convex analysis part 2). This shows that the gradient ∇φ⋆\nabla\varphi^\star∇φ⋆ is Lispchitz-continuous.

Let φ\varphiφ be a differentiable, μ\muμ-strongly convex function on C∘C^\circC∘ then φ⋆\varphi^\starφ⋆ has a gradient that is is 1/μ1/\mu1/μ-Lipschitz continuous over ∇φ(C∘)\nabla \varphi(C^\circ)∇φ(C∘), i.e.:

∥∇φ⋆(u)−∇φ⋆(v)∥≤1μ∥u−v∥,∀u,v∈∇φ(C∘),\begin{array}{c} \| \nabla \varphi^\star(u)-\nabla \varphi^\star(v) \| &\le& {1\over \mu}\|u-v\|,\quad \forall u,v\in\nabla\varphi(C^\circ),\end{array}∥∇φ⋆(u)−∇φ⋆(v)∥​≤​μ1​∥u−v∥,∀u,v∈∇φ(C∘),​ where ∇φ(C∘)={u∈Rn ∣ ∃x∈C∘,u=∇φ(x)}\nabla \varphi (C^\circ)=\{u\in\mathbb R^n\,|\,\exists x\in C^\circ, u=\nabla \varphi(x)\}∇φ(C∘)={u∈Rn∣∃x∈C∘,u=∇φ(x)}.
