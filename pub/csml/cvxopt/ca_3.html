Convex analysis (part 3)

Strict and strong convexity

Remember that for a convex function \(\psi:C\to \mathbb R\), the subdifferential at a point \(x\in C^\circ\) is non-empty and for any subgradient \(y\in\partial \psi(x)\), the following inequality holds: $$\begin{array}{c} \psi(z) &\ge& \psi(x) + \left\langle z-x, y\right\rangle, \quad\forall z\in C. \end{array}$$ This simply means that the graph of the function is above its supporting hyperplanes. When that is strictly the case, the function is said to be strictly convex.

The function \(\psi:C \to \mathbb R\) is said to be strictly convex at \(x\in C^\circ\) if the inequality (1) holds strictly for all \(z\neq x\).

For example \(\psi(x)=x\log(x)\) is strictly convex on \(\mathbb R^+\) whereas \(\psi(x)=|x|\) isn't.

Bregman divergence

Let's consider the particular case where \(\psi\) is strictly convex and differentiable so that \(\partial \psi(x)=\{\nabla \psi(x)\}\). In that case we have $$\begin{array}{c} \psi(z) &\ge& f(x) + \left\langle z-x, \nabla \psi(x)\right\rangle, \quad \forall z\in C, \end{array}$$ with equality if and only if \(z=x\). We can use this to define a notion of similarity between two points \(x\) and \(z\).

Let \(\psi\) denote a strictly convex and differentiable function on \(C\). We define the Bregman-divergence on \(C\times C\) associated with \(\psi\) as $$\begin{array}{c} B_\psi(z, x) &=& \psi(z)-\psi(x)-\left\langle z-x,\nabla \psi(x)\right\rangle.\end{array}$$

Given (2), we have that \(B_\psi(z, x)>0\) for all \(z\neq x\) and \(B_\psi(z, x)=0\) iff \(x=z\) so that \(B_\psi\) is a positive-definite function(al).

For example, \(\psi(x) = \frac12\left\langle x, x\right\rangle\) then $$\begin{aligned} B_\psi(z, x) \quad\!\!&=\quad\!\! \frac12\left\langle z, z\right\rangle - \frac12\left\langle x, x\right\rangle - \left\langle z-x, x\right\rangle\\ &=\quad\!\! \frac12\left\langle z-x, z-x\right\rangle \end{aligned}$$ which is just the squared Euclidean (\(\ell^{2}\)) distance.The factor \(1/2\) might seem irrelevant but it makes other developments a bit nicer so that, traditionally in convex-optimisation, the squared Euclidean distance is always scaled by a factor two.

Strong convexity

The function \(\varphi:C\to \mathbb R\) is said to be \(\mu\)-strongly convex at \(x\in C^\circ\) with parameter \(\mu>0\) if the subgradient inequality (1) holds with:

$$\begin{array}{c} \varphi(z) &\ge& \varphi(x) + \left\langle z-x, y\right\rangle + {\mu\over 2}\|z-x\|_2^2, \end{array}$$ for any \(z\in C\) and \(y\in\partial \varphi(x)\).

Intuitively, a strongly convex function is a strictly convex function whose graph goes away from the supporting hyperplane sufficiently fast (at least quadratically).

Note that if the function \(\varphi\) is \(\mu\)-strongly convex and differentiable, then the Bregman divergence associated to it is lower bounded by the squared \(\ell^{2}\)-distance:

$$\begin{array}{c} B_{\varphi}(z,x) &\ge& {\mu\over 2} {\|z-x\|^{2}_{2}}, \quad\forall x,z \in C. \end{array}$$

Lipschitz continuity and strong convexity

This is a useful result to prove convergence rates of some iterative minimisation methods but can easily be skipped in a first reading.

First we recall that a function \(\phi\) is \(\beta\)-Lipschitz-continuous if \(\forall u,v\in \text{dom}\, \phi\) the following inequality holds:

$$\begin{array}{c} \|\phi(u)-\phi(v)\| &\le& \beta \|u-v\|. \end{array}$$

We will show that if \(\varphi:C\to \mathbb R\) is a differentiable, strongly convex function then \(\nabla \varphi^\star\) is Lipschitz continuous.

Using (6), we can write $$\begin{array}{c} \mu \|x-z\|^{2}_{2} &\le& B_{\varphi}(x,z)+B_{\varphi}(z,x) . \end{array}$$ for any \(x, z\in C^\circ\). Plugging the definition of the Bregman divergence in the right hand side and letting \(u=\nabla \varphi(x)\) and \(v=\nabla \varphi(z)\) we get $$\begin{array}{c} \mu\| x-z\|^{2}_{2} &\le & \langle z-x, u-v \rangle \\ &\le& \|x-z\|_2\|u-v\|_2, \end{array}$$ using Cauchy-Schwartz's inequality. Rearranging terms yields $$\begin{array}{c} \|x-z\|_{2} &\le& {1\over \mu}\| u-v\|_2. \end{array}$$

Since \(x\in C^\circ\) and \(u=\nabla \varphi(x)\), we can write \(x = \nabla \varphi^\star(u)\) and similarly, \(z=\nabla\varphi^\star(v)\) (see convex analysis part 2). This shows that the gradient \(\nabla\varphi^\star\) is Lispchitz-continuous.

Let \(\varphi:C\to\mathbb R\) be a differentiable, \(\mu\)-strongly convex function then \(\varphi^\star\) has a gradient that is is \(1/\mu\)-Lipschitz continuous over \(\nabla \varphi(C^\circ)\), i.e.:

$$\begin{array}{c} \| \nabla \varphi^\star(u)-\nabla \varphi^\star(v) \| &\le& {1\over \mu}\|u-v\|,\quad \forall u,v\in\nabla\varphi(C^\circ), \end{array}$$ where \(\nabla \varphi (C^\circ)=\{u\in\mathbb R^n\,|\,u=\nabla \varphi(x),\, x\in C^\circ\}\).

Remark: we often use \(C^\circ\) instead of just \(C\) to make the presentation simpler, indeed we know that on \(C^\circ\) the subgradients exist and are bounded. Of course these results can be generalised so that we don't have to deal with interiors everywhere but some care is needed to do it properly and the extra details are not particularly useful for this presentation. We refer the interested reader to Rockafellar's book for more details.

Bregman divergence and convex conjugacy

Let \(\psi:C\to\mathbb R\) be a differentiable and strictly convex function and let's consider the Bregman divergence induced by \(\psi^\star\): $$\begin{array}{c} B_{\psi^\star}(u, v) &=& \psi^\star(u) - \psi^\star(v) - \left\langle u-v, \nabla\psi^\star(v)\right\rangle. \end{array}$$ Let us now consider \(u, v\in\mathbb R^n\) such that \(u=\nabla \psi(x)\) and \(v=\nabla\psi(z)\) for some \(x, z\in C^\circ\). We then have \(\nabla^\star(\nabla \psi (z)) = z\) (cf. convex analysis part 2). Also, since \((x, \nabla \psi(x))\) is a dual pair, we have $$\begin{array}{c} \psi(x) + \psi^\star(\nabla\psi(x)) &=& \left\langle x, \nabla \psi(x)\right\rangle, \end{array}$$ and the same for \(z\). Plugging this into (12), we obtain $$\begin{array}{c} B_\psi^\star(\nabla\psi(x), \nabla\psi(y)) &=& \psi(z)-\psi(x)-\left\langle z-x, \nabla \psi (x)\right\rangle. \end{array}$$

Let \(\psi:C\to \mathbb R\) be a differentiable, strictly convex function. Then for \(x, z \in C^\circ\) we have $$\begin{array}{c} B_{\psi^\star}(\nabla \psi(x), \nabla\psi(z)) &=& B_\psi(z, x). \end{array}$$

Of course, by symmetry, we also have $$\begin{array}{c} B_{\psi}(\nabla \psi^{\star}(u),\nabla\psi^{\star}(v)) &=& B_{\psi^{\star}}(u,v), \end{array}$$ for any \(u, v\in\nabla\psi(C)\).
