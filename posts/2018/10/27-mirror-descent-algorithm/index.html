From Euclidean to arbitrary projections

In the notes on the projected gradient descent (PGD) we showed that a way to solve the constrained minimisation problem with a differentiable fff is to follow the iteration

xk+1 ⁣ ⁣= ⁣ ⁣πC(xk−αk∇f(xk)), x_{k+1} \quad\!\! =\quad\!\! \pi_C(x_k - \alpha_k \nabla f(x_k)), xk+1​=πC​(xk​−αk​∇f(xk​)),

where πC\pi_CπC​ denotes the Euclidean projection onto CCC. By definition of the Euclidean projection, this can also be written as

xk+1 ⁣ ⁣= ⁣ ⁣arg⁡min⁡x∈C ∥(xk−αk∇f(xk))−x∥22= ⁣ ⁣arg⁡min⁡x∈C ∥(x−xk)+αk∇f(xk)∥22.\begin{aligned} x_{k+1} \quad\!\!&=\quad\!\! \arg\min_{x\in C}\,\, \|(x_k-\alpha_k\nabla f(x_k))-x\|_2^2 \\ &=\quad\!\! \arg\min_{x\in C} \,\,\| (x-x_k) + \alpha_k\nabla f(x_k)\|_2^2.\end{aligned}xk+1​​=argx∈Cmin​∥(xk​−αk​∇f(xk​))−x∥22​=argx∈Cmin​∥(x−xk​)+αk​∇f(xk​)∥22​.​

Now, since the ℓ2\ell^2ℓ2-norm is induced by the inner product (i.e. ⟨x,x⟩=∥x∥22\left\langle x, x\right\rangle = \|x\|_2^2⟨x,x⟩=∥x∥22​), we can use the decomposition

∥(x−xk)+αk∇f(xk)∥22 ⁣ ⁣= ⁣ ⁣∥x−xk∥22+2αk⟨x,∇f(xk)⟩+Mk, \|(x-x_k) + \alpha_k\nabla f(x_k)\|_2^2 \quad\!\! =\quad\!\! \|x-x_k\|_2^2 + 2\alpha_k\left\langle x, \nabla f(x_k)\right\rangle + M_k, ∥(x−xk​)+αk​∇f(xk​)∥22​=∥x−xk​∥22​+2αk​⟨x,∇f(xk​)⟩+Mk​,

where MkM_kMk​ does not depend on xxx (and can thus be ignored). Rearranging terms, we're then left with the equivalent iteration:

xk+1 ⁣ ⁣= ⁣ ⁣arg⁡min⁡x∈C {⟨x,∇f(xk)⟩+1αk∥x−xk∥222}. x_{k+1} \quad\!\! =\quad\!\! \arg\min_{x\in C}\,\, \left\{\left\langle x, \nabla f(x_k)\right\rangle + {1\over \alpha_k} {\|x-x_k\|_2^2\over 2}\right\}. xk+1​=argx∈Cmin​{⟨x,∇f(xk​)⟩+αk​1​2∥x−xk​∥22​​}.

This new way of writing the PGD allows two important comments:

  1. the objective shows how the PGD corresponds to a tradeoff between following the direction of the negative gradient (first term) and not moving too much from the current point (second term) while staying in CCC,

  2. the second term is an isotropic measure of distance from the current point xkx_kxk​, but what if we used another measure of distance?

The first point might be obvious to you if you're already seen the gradient descent and related methods but it's actually a bit deeper than what it may seems on a cursory glance, I discuss this more in thoughts on first order methods.

The second point gives the generalised projected gradient descent.

The generalised projected gradient descent (GPGD) is defined for any positive-definite function d:Rn×Rn↦R+d:\mathbb R^n\times\mathbb R^n\mapsto\mathbb R^+d:Rn×Rn↦R+ by the following iteration:

xk+1 ⁣ ⁣= ⁣ ⁣arg⁡min⁡x∈C {⟨x,∇f(xk)⟩+1αkd(x,xk)}. x_{k+1} \quad\!\! =\quad\!\! \arg\min_{x\in C}\,\, \left\{\left\langle x, \nabla f(x_k)\right\rangle + {1\over \alpha_k} d(x, x_k)\right\}. xk+1​=argx∈Cmin​{⟨x,∇f(xk​)⟩+αk​1​d(x,xk​)}.

Those positive-definite functions are sometimes called divergences and verify d(u,v)>0d(u, v)>0d(u,v)>0 for u≠vu\neq vu=v and d(u,u)=0d(u, u)=0d(u,u)=0.

One reason why one might want to consider another distance-like function to penalise how much we move in a particular direction is that doing so can better reflect what we may know about the geometry of CCC which can make steps easier to compute or accelerate convergence to a minimiser.

For a given divergence, there now remains to compute the corresponding iteration steps which may be intractable depending on ddd. But there happens to be a popular class of divergences for which it is tractable and which we have already discussed: the Bregman divergences.

Bregman divergences and the mirror descent algorithm

Recall that for a strictly convex and differentiable function ψ\psiψ, we can define the Bregman divergence associated with ψ\psiψ as a positive-definite function BψB_\psiBψ​ with

Bψ(x,y) ⁣ ⁣:= ⁣ ⁣ψ(x)−ψ(y)−⟨x−y,∇ψ(y)⟩. B_\psi(x, y) \quad\!\! :=\quad\!\! \psi(x)-\psi(y)-\left\langle x - y, \nabla \psi(y)\right\rangle. Bψ​(x,y):=ψ(x)−ψ(y)−⟨x−y,∇ψ(y)⟩.

Let us now consider a μ\muμ-strongly convex and differentiable function φ\varphiφ and the associated Bregman divergence BφB_\varphiBφ​. If we use this as divergence in (5), the GPGD iteration is

xk+1 ⁣ ⁣∈ ⁣ ⁣arg⁡min⁡x∈C {⟨x,∇f(xk)⟩+1αkBφ(x,xk)} x_{k+1} \quad\!\! \in\quad\!\! \arg \min_{x\in C} \,\,\left\{ \left\langle x, \nabla f(x_k)\right\rangle + {1\over \alpha_k}B_\varphi(x, x_k)\right\} xk+1​∈argx∈Cmin​{⟨x,∇f(xk​)⟩+αk​1​Bφ​(x,xk​)}

for αk>0\alpha_k>0αk​>0. It's straightforward to take the subgradient of the objective and write the FOC for one step:

0 ⁣ ⁣∈ ⁣ ⁣αk∇f(xk)+∇φ(xk+1)−∇φ(xk)+NC(xk+1), 0 \quad\!\! \in\quad\!\! \alpha_k\nabla f(x_k) + \nabla \varphi(x_{k+1}) - \nabla \varphi (x_k) + N_C(x_{k+1}), 0∈αk​∇f(xk​)+∇φ(xk+1​)−∇φ(xk​)+NC​(xk+1​),

which, after rearranging terms, reads

xk+1 ⁣ ⁣∈ ⁣ ⁣(∇φ+NC)−1(∇φ(xk)−αk∇f(xk)). x_{k+1} \quad\!\! \in\quad\!\! (\nabla\varphi + N_C)^{-1} (\nabla \varphi(x_k) - \alpha_k \nabla f(x_k)).xk+1​∈(∇φ+NC​)−1(∇φ(xk​)−αk​∇f(xk​)).

To complete this, observe that (∇φ+NC)≡∂(φ+iC)(\nabla \varphi + N_C) \equiv \partial (\varphi + i_C)(∇φ+NC​)≡∂(φ+iC​) so that letting ϕ≡φ+iC\phi \equiv \varphi + i_Cϕ≡φ+iC​, and using that for h∈Γ0h\in \Gamma_0h∈Γ0​, (∂h)−1≡∂h⋆(\partial h)^{-1}\equiv\partial h^\star(∂h)−1≡∂h⋆ (cf. convex analysis part 2), we end up with the mirror descent algorithm.

The mirror descent algorithm (MDA) is the iteration

xk+1 ⁣ ⁣= ⁣ ⁣∇ϕ⋆(∇φ(xk)−αk∇f(xk)) x_{k+1} \quad\!\! =\quad\!\! \nabla\phi^\star (\nabla \varphi(x_k) - \alpha_k\nabla f(x_k)) xk+1​=∇ϕ⋆(∇φ(xk​)−αk​∇f(xk​))

where ϕ⋆(y)=sup⁡z∈C[⟨z,y⟩−φ(z)]\phi^\star(y) = \sup_{z\in C}[\left\langle z,y\right\rangle-\varphi(z)]ϕ⋆(y)=supz∈C​[⟨z,y⟩−φ(z)].

Note that ϕ\phiϕ is also strongly convex on CCC so that it is differentiable which is why we can write ∇ϕ⋆\nabla \phi^\star∇ϕ⋆ (its gradient is even Lipschitz as we showed in convex analysis part 3).

A final note is that, as for the PGD, the differentiability of fff can be relaxed to sub-differentiability without much changes, the iteration is then xk+1=∇ϕ⋆(∇φ(xk)−αkf′(xk))x_{k+1}=\nabla\phi^\star(\nabla \varphi(x_k)-\alpha_k f'(x_k))xk+1​=∇ϕ⋆(∇φ(xk​)−αk​f′(xk​)) with f′(xk)∈∂f(xk)f'(x_k)\in \partial f(x_k)f′(xk​)∈∂f(xk​).
