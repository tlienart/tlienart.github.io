Projected gradient descent

Here we will show a general method to approach a constrained minimisation problem of a convex, differentiable function fff over a closed convex set C⊂RnC\subset \mathbb R^nC⊂Rn . Such problems can be written in an unconstrained form as we discussed in the introduction

min⁡x ⁣ ⁣f(x)+iC(x),\begin{array}{c} \min_x\quad\!\! f(x) + i_C(x),\end{array}minx​f(x)+iC​(x),​

where iCi_CiC​ is the indicator function associated with the set.

To get to the method, we will need three main parts:

  1. discuss the first order condition (FOC) of the problem which will bring us to the notion of normal cone,

  2. discuss how the normal cone relates to the Euclidean projection,

  3. introduce the projected gradient descent algorithm.

First order condition

The FOC indicates that x†∈Cx^\dagger \in Cx†∈C is a minimiser if and only if 0∈∂(f+iC)(x†)0\in \partial(f + i_C)(x^\dagger)0∈∂(f+iC​)(x†) . iC(z)≥iC(x)+⟨z−x,y⟩,∀z.\begin{array}{c} i_C(z) &\ge& i_C(x) + \left\langle z-x, y\right\rangle, \quad\forall z. \end{array}iC​(z)​≥​iC​(x)+⟨z−x,y⟩,∀z.​

Normal cone and Euclidean projection

A useful (and obvious) property of the normal cone which we shall use in the sequel is that it is invariant to a non-negative scalar multiplication, i.e.:

if u∈NC(x) then αu∈NC(x), ∀α≥0.\begin{array}{c} \mathrm{if}\,\, u\in N_C(x)\,\,\mathrm{then}\,\, \alpha u\in N_C(x), \,\forall \alpha \ge 0. \end{array}ifu∈NC​(x)thenαu∈NC​(x),∀α≥0.​

We will now show the connection between the normal cone and the Euclidean projection on CCC . Remember that the Euclidean projection onto CCC (denoted πC\pi_CπC​ ) is defined as follows for any z∈Rnz\in \mathbb R^nz∈Rn :

πC(z) ⁣ ⁣= ⁣ ⁣arg⁡min⁡x∈C 12∥x−z∥22 \pi_C(z) \quad\!\! =\quad\!\! \arg\min_{x\in C}\,\, \frac12\|x-z\|_2^2 πC​(z)=argx∈Cmin​21​∥x−z∥22​

Note that in the unconstrained problem, C=RnC=\mathbb R^nC=Rn and hence πC=I\pi_C=\mathbb IπC​=I , the identity operator. Note also that the factor 1/21/21/2 is here for aesthetics when computing the gradient which is just (x−z)(x-z)(x−z) .

Considering the condition (5) for the optimisation problem (7), we have that if

This is equivalent to requiring z∈x†+NC(x†)z \in x^\dagger + N_C(x^\dagger)z∈x†+NC​(x†) or, equivalently, z∈(I+NC)(x†)z\in (\mathbb I + N_C)(x^\dagger)z∈(I+NC​)(x†) . Replacing x†x^\daggerx† by πC(z)\pi_C(z)πC​(z) and re-arranging terms gives

This is a classical and important relationship.

Projected gradient descent

We indicated at (6) that for any α≥0\alpha \ge 0α≥0 and u∈NC(x)u\in N_C(x)u∈NC​(x) , then αu∈NC(x)\alpha u\in N_C(x)αu∈NC​(x) . We can rewrite this as:

(x+αu)−x∈NC(x),\begin{array}{c} (x+\alpha u) - x &\in& N_C(x), \end{array}(x+αu)−x​∈​NC​(x),​

which may seem pointless but will lead us to a fixed-point algorithm (remember from the introduction that many algorithms for convex optimisation can be expressed in terms of fixed point algorithms).

Let z:=(x+αu)z:=(x+\alpha u)z:=(x+αu) so that (11) can be rearranged to z∈(I+NC)(x)z\in (\mathbb I+N_C)(x)z∈(I+NC​)(x) . By (10), this is equivalent to x=πC(z)x=\pi_C(z)x=πC​(z) or

x=πC(x+αu),(α≥0, u∈NC(x)).\begin{array}{c} x &=& \pi_C(x+\alpha u), \quad (\alpha \ge 0, \,u\in N_C(x)). \end{array}x​=​πC​(x+αu),(α≥0,u∈NC​(x)).​

To finish up, let's go back once more to the FOC (5) which indicates that if x†x^\daggerx† is such that −∇f(x†)∈NC(x†)-\nabla f(x^\dagger) \in N_C(x^\dagger)−∇f(x†)∈NC​(x†) then it is a minimiser. Combined with (12), we get the following useful fixed-point form for the minimiser of the constrained problem:

x† ⁣ ⁣= ⁣ ⁣πC(x†−α∇f(x†)). x^\dagger \quad\!\! =\quad\!\! \pi_C(x^\dagger - \alpha \nabla f(x^\dagger)). x†=πC​(x†−α∇f(x†)).

Note: had we not assumed that fff was differentiable on CCC , we would still have that there exists a subgradient f′(x†)∈∂f(x†)f'(x^\dagger) \in \partial f(x^\dagger)f′(x†)∈∂f(x†) with −f′(x†)∈NC(x†)-f'(x^\dagger)\in N_C(x^\dagger)−f′(x†)∈NC​(x†) and consequently the fixed-point x†=πC(x†−αf′(x†))x^\dagger = \pi_C(x^\dagger - \alpha f'(x^\dagger))x†=πC​(x†−αf′(x†)) leading the projected subgradient method. In practice however, one tries to cast a convex objective function as a sum of a smooth function with a non-smooth one, to follow a gradient descent on the smooth part and use a projection for the non-smooth part (this is the case here with iCi_CiC​ being the non-smooth part).

The algorithm

Well there's not much left to do. Applying a fixed-point iteration to (13) from a starting point x0x_0x0​ generates the sequence {x0,x1,… }\{x_0, x_1, \dots\}{x0​,x1​,…} via

xk+1=πC(xk−αk∇f(xk))\begin{array}{c} x_{k+1} &=& \pi_C(x_k - \alpha_k \nabla f(x_k)) \end{array}xk+1​​=​πC​(xk​−αk​∇f(xk​))​

where αk>0\alpha_k > 0αk​>0 is the step size (ignoring the trivial case αk=0\alpha_k=0αk​=0 ). This is the projected gradient descent method.

Assuming that the αk\alpha_kαk​ are picked sensibly and basic regularity conditions on the problem are met, the method enjoys a convergence rate (f(xk)−f(x†))=O(k−1)(f(x_k)-f(x^\dagger)) = \mathcal O(k^{-1})(f(xk​)−f(x†))=O(k−1) (see references for more).

Note: this method is pretty easy to apply provided you can compute the projection. Of course, there may be situations for which computing the projection is as hard as the initial problem! But there are many special cases where efficient projection can be applied (e.g. if CCC is a polyhedron, i.e. corresponds to a set of xxx such that Ax≤bAx\le bAx≤b for some matrix AAA and vector bbb ). See for example Liu and Ye (2009).
